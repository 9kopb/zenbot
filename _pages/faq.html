What Is Zenbot?

Zenbot is an online natural language processing service and chat bot hosting.

What Can Zenbot Do?

Zenbot allows you to create Natural Language driven user interface for your application, website or messenger chat bot. So that you can create Siri-like complex dialog interface for any type of application or website and accept user's text or even voice input.

How Does It Work?

Zenbot matches each text request through a set of currently active patterns in your Botscript. A matched pattern can activate a set of actions in the context which will perform some stuff. Then Zenbot returns one of the text output messages corresponding to this context, so the user is walking through a natural language dialog.

How Much Does It Cost?

Zenbot is free even for commercial use. No rate limits as well.

How Do I Get Started?

To make a new bot you have to create a Botscript which contains all data including the dialog with the user and actions Zenbot must perform.

What Do I Need to Know before Starting?

All you need to get started is to read about Botscript and look at some of our samples. You do not have to have any skills in programming or artificial intelligence disciplines.

How to Create a Botscript?

A Botscript is a simple XML formatted file. You can create it with any text editor you like, for exaple, Sublime Text or Windows Notepad.

How to Upload the Botscript to Zenbot?

Just go to the web console and create a new bot. Then you will be prompted to upload your Botscript file.

How to Use Zenbot from My Application or Website?

Zenbot provides a REST API to allow you to make simple HTTP calls to integrate your application or website with your bot.

How to Integrate Zenbot with My Facebook, Slack or Telegram Bot?

Zenbot is already integrated with such messenger platforms as Facebook, Slack and Telegram. So, all you need to bind your Zenbot bot to these messengers is to provide your credentials in the web console.

Which Languages Are Supported?

Strictly speaking, Zenbot supports any language thanks to the flexible patterns syntax. So you can add user’s phrases in any language you wish.

But some of general concepts (like Numbers, Date and Time handling) are built in Zenbot and support only a particular set of languages, such as English and Russian.

All of them are open source and you can contribute to and extend these items with your language as well.

What Are the Differences between Zenbot and Wit.ai?

There are three major differences between Zenbot and Wit.ai:

  • Zenbot implements a full stack of NLP techniques, not only user’s intent resolution.

    This means that you do not have to implement a user-to-machine conversation logic on your side. Zenbot proceeds each request, manages contexts, performs actions, evaluates scripts, collects and stores variables and generates text output. Thus the client system just sends a text request and receives a complete response.

  • Zenbot is based on the pattern matching mechanism instead of deep learning "magic".

    Pattern matching provides an NLP flexibility together with full control of the dialog system. Thus Zenbot takes data already prepared by a human and works as expected.

  • Zenbot doesn’t provide any web interface for dialog structure designing.

    Such web interfaces are often more confusing than useful if you deal with complex dialogs. Thus Zenbot defines a simple XML file format where you can describe such a dialog and manage it as the source code for your bot.
